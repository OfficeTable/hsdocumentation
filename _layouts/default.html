1. Installation

1.1 Version

Current Version (Latest Update): core1 Beta 7

Current Major Version: core1

1.2 Installation and Uninstallation

To Install, Download HomeshellSetup.exe and Run it. Accept the terms, and Start Installlation.

To Uninstall, Go to Windows Settings, then Click on Apps. Go to Apps and Features and look for Homeshell.core. Just Click Uninstall, Comfirm it in the uninstaller, and Start Uninstallation.

1.3 Updates

In Homeshell, Go to Homeshell or Options, Click Updates, Complete the Form to check if Updates are Available, then Download the new setup from the Link and Follow the Installation Apps in 1.1.

3.2 Browser Troubleshooting

 

Error/Issue Steps to Resolve

Not Able to Connect

or

Not Connected to Internet

1. Make sure your Connected to the internet

2. Run Windows Internet Diagnostics If Connected

3. Review/Change Proxy Settings If Diagnostic Fails

4. Turn Proxy OFF as a Last Resort
Page Crashed

1. Retry

2. Complete Steps 1-3 Here If Retrying Fails

3. Free Up Memory If Steps Fail
Stuck on Unwanted Page

1. Go Back

2. Close WebBox if Going Back Fails

4.2 Client Troubleshooting

 

Error/Issue Steps to Resolve
"Banned Messege" You have been Suspended. Email Homeshell Support.

2. Reopen the Client if Retry Fails

3. Reopen Homeshell if Client Reopen Fails

4. Wait. The Server might have had downtime
